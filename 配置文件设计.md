# 功能点

- 定时执行任务列表
- 是否监听某端口等待外部连接
- 是否定时更新请求服务器更新配置
- 是否反弹shell

# 设计

```json
{
    //配置文件版本号
    "version":"1.0",

    //定时执行命令
    "cmdlines": ["cmd1","cmd2"],
    "cmdIntervaltime": 123,//单位秒

    //监听端口,若不启用此功能，则设置为-1
    "Listening_port": -1,

    //定时请求服务器更新配置文件,若不启用此功能，则设置为-1
    "updata_time":-1,//单位秒

    //反弹shell 
    "enable_shell":1,
    "shell_host":"127.0.0.1",
    "shell_port" : "12",
    "shell_protocol":"TCP",//使用协议
    "retry_time":10//重试连接间隔时间

}
```
